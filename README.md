# [Alucard][1]

Info about github repo specific details

## Setup:

```
 npm install
 cp server/config_example.js server/config.js
 cp config_example.js config.js
```

## Run the application:

```
 npm start
```

or if nodemon is installed:

```
 nodemon
```
